# Tuple tests

The tests in this directory, except for the tests in the `tuple.extra` folder,
are derived from llvm's [libcxx test
suite](https://github.com/llvm/llvm-project/tree/main/libcxx/test/std/utilities/tuple),
they have been modified to work on GPU using Kokkos. The original copyright
notice can be found in [ORIGINAL_LICENSE.txt](./ORIGINAL_LICENSE.txt).
